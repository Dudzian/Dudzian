# Archive

This directory przechowuje wyłącznie artefakty dokumentacyjne związane z dawną warstwą
`KryptoLowca`.  Po pełnym usunięciu kodu archiwalnego nie utrzymujemy tu już żadnych shimów ani
modułów wykonywalnych – aktywny rozwój i wszystkie biblioteki runtime znajdują się w
`bot_core`.  Zawartość katalogu służy jedynie celom historycznym i migracyjnym.
