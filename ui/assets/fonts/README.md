# Font Awesome Free

Dane czcionki Font Awesome 6 Free (wariant Solid, plik pierwotny `FontAwesome6Free-Solid-900.otf`) pochodzą z projektu
[Font Awesome Free](https://fontawesome.com/) i są dystrybuowane na licencji CC BY 4.0 / SIL OFL 1.1 (w zależności od
wariantu).

Aby uniknąć przechowywania binarnych plików w repozytorium, font jest osadzony w postaci tekstowego ciągu Base64
(`ui/qml/design-system/fonts/FontAwesomeData.js`), który jest dynamicznie ładowany przez QML i PySide6.
