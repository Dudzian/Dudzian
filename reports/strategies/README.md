# Raporty walidacji strategii

Pliki w tym katalogu są generowane przez `StrategyQualityPipeline`. Każdy
scenariusz sandboxowy posiada osobny raport JSON zawierający:

* parametry strategii,
* metryki kalibracyjne,
* wynik oceny względem progów akceptacyjnych.

Podsumowanie z ostatniego przebiegu pipeline'u jest zapisywane w pliku
`summary.json`.
